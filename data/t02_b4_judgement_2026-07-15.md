## 1. 判定（事前登録 3 項目）

| 項目 | 事前登録の条件 | 結果 | 判定 |
|---|---|---|---|
| 合成 > 単一 | 合成の CV MAE < 最良単一因子・permutation p<0.05 | mult 1.012 < 単一最良 A_mass 1.185・合成の perm p=0.004 | **成立 → kill 基準は非発動** |
| 機構 > 無機構 | 三因子系の最良が全ベースラインに勝つ | 機構最良 1.012 > **log₁₀(params) 0.801** | **不成立 → 「機構測定の付加価値なし」と報告** |
| 乗法 vs 加法 | MAE 差を報告（考察） | 乗法 1.012/1.128 ≪ 加法 1.335/1.703 | 乗法が明確に優位（関連理論の傍証としての解釈は考察・§5） |

**Part B の中心主張（短文脈の機構因子合成が実効文脈長をよく予測する）は不成立**。合成は単一因子には勝つ（kill 非発動）が、モデルサイズ 1 変数（log₁₀ params・MAE 0.801 ≈ 実効長を平均 1.7 倍以内で当てる）に負ける。Part A は設計どおり独立に生存。

## 2. データセット（N=15・6 families）

y = log₂(effective)。effective の出典 = 自前 7 点（B3 判定書・answer_prefix 復元プロトコル）+ Qwen3 4b/8b/14b（公開行の閾値法再計算・author-reported・32K/32K/64K）+ 横断 5 点（NVIDIA 測定行の再計算・llama-3.1-8b 32K・mistral-7b 16K・mistral-nemo 16K・yi-34b 32K・phi3-mini 32K）。censored 3 点（qwen3-0.6b・gemma-270m/1b <4K）は規約どおり 2048 代入（主）+ 除外（感度）。>128K 点はなし。横断 8→5 の減は B2 断念（phi3-medium・GLM×2）による。

因子: R（attention エントロピー成長率・全 15 点）・A_mass / A_gini（retrieval head 総質量・Gini・全 15 点）・W = **全層平均 value erank（主・全 15 点）**。

## 3. 主結果（LOFO-CV MAE・log₂ 単位・N=15）

| モデル | MAE | 区分 |
|---|---|---|
| **log₁₀ params** | **0.801** | ベースライン（最良） |
| log₂ claimed | 0.979 | ベースライン |
| mult R×A_gini×W | 1.012 | 機構最良 |
| mult R×A_mass×W | 1.128 | 機構 |
| A_mass 単一 | 1.185 | 単一最良 |
| d_model | 1.206 | ベースライン |
| add R+A_mass+W | 1.335 | 機構 |
| R 単一 | 1.382 / W_erank 単一 1.524 / A_gini 単一 1.541 / add R+A_gini+W 1.703 | |

permutation（1000 回・seed 固定）: 機構最良の自体の予測有意性 p=0.004・単一最良 p=0.014。**より厳しい paired 差分検定（事後・開示）**: (合成−単一) p=0.139・(合成−ベースライン) p=0.665——合成が単一に勝つ「差」自体は N=15 では個別有意でない。事前登録の p<0.05 は §4 の定義（目的変数シャッフルによる CV MAE の帰無分布 = 合成自体の有意性）で判定した。

**機構因子がサイズの代理になっている**のが敗因の構造: corr(log₁₀ params, ·) = A_mass 0.86・W_erank 0.52・R 0.42、かつ corr(log₁₀ params, y) = 0.89。因子は y と相関する（A_mass 0.75・W 0.68）が、その相関のほとんどをサイズが媒介しており、サイズを超える増分情報を持たない。

## 4. 感度分析

1. **censored 3 点除外（N=12）**: log₂ claimed 0.659 が最良・mult R×A_mass×W 0.760・log₁₀ params 0.779。機構がベースライン束の一部（params）に勝つ入れ替わりはあるが、**最良ベースライン（claimed）には勝てない**——主判定不変。
2. **W = NER（正規化 erank）**: 単一 2.10・加法 1.92——生 erank より一貫して悪い。主変種選択（生 erank）に不利な逆転なし。
3. **W = band 平均（健全 band 10 点のみ）**: band 平均 2.59 vs 全層平均 2.89（同一 10 点）——band 平均がやや良いが N 縮小でともに劣化。主変種（全層平均・15 点）の判定を変えない。
4. **W = k\*_J（階段 9 点・14b 除外・2 families = leave-one-series-out）**: k\*_J 単一 1.37 は同条件の erank 4.02 より良い——ただし k\*_J はサイズとともに**減少**する（Part A の J-lens 劣化）ため、実効長との関係は負の相関を経由したサイズ代理であり、機構的な読みは不可（開示のみ）。
5. **Gemma-3 を 85.6 で切った y（閾値混在の感度・指示書 §7 裁定 1）**: Gemma の段構造が消失した縮退 y の下では mult 1.367 < params 1.574 と逆転するが、この y は B3 判定書 §3.2 のとおり測定の実態（-pt checkpoint）に合わない縮退形——主判定の反証には使わない（開示のみ）。

## 5. 考察（判定外）

- **乗法 ≫ 加法**は両 A 変種で一貫（1.01 vs 1.34・1.13 vs 1.70）。「因子の対数和（= 積）が実効長の対数を運ぶ」形は、資源の積型合成という関連理論の方向と整合する——ただし本判定は「機構因子はサイズに勝てない」なので、傍証としての重みは弱い。
- **報告価値の重心**: (i) 「短文脈機構因子は長文脈実効長の予測でサイズ 1 変数に勝てない」という否定的結果そのもの（コスト正当化の反証——因子測定は RULER 実測より安いが、サイズを知っているなら不要）(ii) B3 の effective ladder 自体（Gemma-3 の RULER 値は公開初）(iii) 測定器の科学（Part A からの継続 = J-lens 劣化・band 規則の両端故障・answer_prefix 回帰の定量）。

## 6. 開示事項

1. **プロトコル混在**: 自前 7 点 = answer_prefix 復元・NVIDIA 5 点 = 公式ハーネス（プロトコル同一性は外部から確定不能・B3 判定書 開示 5）・Qwen3 3 点 = author-reported（条件同一性未確認・RULER パック §5）。
2. **閾値混在**: Qwen3/横断 85.6・Gemma-3（-pt）79.4。感度 = §4-5。
3. **Gemma R は global 層限定**: R probe の H が SWA 層で系列長 > sliding window のとき NaN → 全長で有限な層（= global attention 層・6 の倍数層）のみで R を再計算（`b4_regression.py` 内・full-attention モデルでは記録値と一致することを assert で保証）。**訂正（同日）**: 当初「0·log0 未ガード」と記述したが、probe には CLIP_FLOOR clamp が既に存在し、NaN 層の clip_count が 0 であることから **NaN は attention 重み自体に含まれている**（transformers の eager 実装が SWA 層・window 超で非有限値を返す——機構の一次特定は B5 Job 1 の診断で行う）。probe の罪は「非有限入力の未ガード」であり、head-row 単位の除外 + 計数を実装済み（2026-07-15・合成 NaN 4 ケースの単体検証済み・B5 指示書 §1）。生 attention は非保存のため B4 時点の救済（global 層限定）は不変。**再訂正（2026-07-17・B5 Job 1 診断で確定・3 訂目）**: 上の 07-15 訂正は誤り。真因は測定コード側——`attn_weights` の実 dtype は bf16 で、`CLIP_FLOOR=1e-45` が bf16 キャストで 0.0 になりクランプが無効化されていた（sliding window 外の列 = softmax 後の厳密ゼロ → log(0) → 0·(-inf)=nan。ゼロ行は有限なので 07-15 の non-finite ガードでは検出不能）。つまり**原記述「0·log0 未ガード」が機能的には正しかった**（ガードは存在したが不活性 = 実質未ガード）。修復 = CLIP_FLOOR 1e-38 + dtype 表現可能性の fail-closed assert（B5 指示書 §6 Amendment 1・受け入れゲート G1-G4 つき）。B4 の救済（global 層限定 R）と判定は不変。
4. **W の主変種は設計の「band 平均」から「全層平均」へ変更（データ強制・事前でなく判定時の裁定)**: 横断 4 点中 3 点（llama/mistral-7b/nemo）で band 規則が故障（run 被覆率 97-100%・band が最前部に落ち k\*logit ≈ 0——qwen3-14b と同型）。band 平均を主にすると横断層の W が系統的に壊れるため全層平均を主・band 平均を健全部分集合の感度に降格。**band 規則の「長すぎ側」故障が Qwen 系以外（Llama/Mistral）でも再現した**のは Part A 教訓の外的一般化として独立の報告価値。
5. **最良合成の選択（4 変種中）は事後**——合成 4 変種すべてのMAE を §3 に全数掲示し、判定は「最良合成 vs 最良単一 vs 最良ベースライン」の構造で行った（変種間の選択的報告はしていない）。
6. **N=15・6 families の検出力限界**: paired 差分 p=0.139/0.665 が示すとおり、MAE 差 0.2-0.4 級は個別有意にならない規模。判定はこの限界の中で事前登録条件を機械適用した結果である。
7. ベースライン「訓練長」は情報源がモデル間で一様に確定できず未実装（claimed length が部分的に代理・開示）。
8. y と A 因子の測定は独立（A = 4K NIAH・y = RULER 13 タスク）だが、両者とも「長文検索」の行動面を含む——循環回避は設計どおり長さ帯の分離（因子 ≤4K・y の判定域 ≥8K）に依存している。

## 7. Part B の結論と次

- **kill 基準（合成 > 単一の不成立）は非発動**。だが中心主張（機構 > 無機構）は不成立——Part B は「否定的結果 + 測定器の科学」として報告書化するのが正直な出口。
- 次work: (1) Part A + Part B の統合報告書化（T02 の出口・Part A HTML と同型の検証済み報告）(2) Oolong 副目的変数（設計上は任意・本判定の後では優先度低下——「サイズに勝てない」構図が Oolong で逆転する見込みは薄い）(3) J-lens 劣化機構・band 規則故障の一般化は関連課題の open のまま。
