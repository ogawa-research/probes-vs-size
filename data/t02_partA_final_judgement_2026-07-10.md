## 2. qwen3-14b の除外裁定 — band 規則の適用前提が不成立

**現象**: 機械規則（隣接 CKA ≥0.9 の最長 run の先頭 floor(L/3) 層）が 14b で band=[1-13]（深度 2.6-33% の最前部）を出力し、k\*_64 = 0.55——他 8 点（6.7-8.8）から桁外れ。logit-lens（0.012）・band4[1-4]（0.000）も同時に死んでおり、**「最前部の層には語彙読み出し可能な表現がまだない」ことを 3 測定器が一致して示す**。lens・モデル・測定の故障ではなく band 選択の故障。

**原因診断（CKA 生データの検分・研究者）**: 隣接 CKA の分布はモデルで大きく違う——

| モデル | 隣接 CKA median | ≥0.9 の割合 | longest run 被覆率 |
|---|---|---|---|
| qwen3-14b | 0.978 | 37/39 | **38/39 = 97%** |
| qwen3-8b | 0.974 | 30/35 | 21/35 = 60% |
| gemma-3-12b | 0.937 | 27/47 | 22/47 = 47% |
| gemma-3-4b | 0.901 | 18/33 | ~33% |

**Qwen 系は隣接 CKA の中央値が 0.97-0.98 と閾値 0.9 の上に張り付いており、run の切れ目は「たまたま 0.9 を割る 1-2 箇所の谷」に依存していた**（8b は層 4-5 の谷 0.81 が run を層 14 から始めていた）。14b でこの前部の谷が 0.9 を超えた（層 1-2 以降すべて ≥0.91）瞬間、run が層 1 まで伸びて band が最前部に落ちた。つまり 14b 固有の異常ではなく、**固定閾値 0.9 の弁別力が Qwen 系で構造的に薄かったことの露呈**（Gemma 系は median 0.90-0.94・谷が多く、規則が意図どおり機能する）。

**裁定**:
1. **main 回帰から除外**。除外基準は測定値でなく CKA 構造で定義する: **longest run の被覆率（run 長/(L−1)）> 90% の場合、CKA ブロック構造が不在で band 規則の適用前提が不成立とみなす**（14b = 97%・他 8 点は最大 60%）。この基準が事後定義であることは開示する（規則設計時に「区間が短すぎる」端点〔270m・閾値緩和で対処〕は想定したが「区間が長すぎる＝弁別不能」端点を見落とした——規則の設計穴）。
2. 14b 込み 9 点回帰は γ̂ = −0.61±0.45（残差が 14b に支配され回帰が壊れる）——参考として開示のみ。
3. **規則改訂は本測定内ではしない**（事前登録維持）。将来の改訂候補（分位ベース閾値・谷検出）は §6 に記録。
4. 14b の d=5120 点を回収する唯一の方法は「深度転写 fallback band（他 Qwen の band 深度 39-79% を 40 層に射影 ≈ 層 16-30）での再測定」——事後 band 選択のため**参考点扱い**（主判定に入れない）。追加取得の判断は §7。

## 3. 本判定（8 点・規約どおり）

**主判定（k=64・main band・プール回帰〔系列別切片・共通勾配〕・df=5）**:

> **γ̂ = −0.120、SE = 0.048、95% CI = [−0.242, +0.003] → 3 出口の「決定不能」＝実額報告**

- **H_G（成長）の棄却は確定**: CI 上限 +0.003 は ε = 0.15 の 1/50。γ = 0.5（√d 比例）とは ~13σ、γ = 1 とは ~24σ の分離。「J-space 容量はモデルサイズで増える」はこのデータで完全に否定された。
- **H_C（厳密な保存 γ=0）も支持しきれない**: CI 上限がゼロにちょうど接する（+0.003）。「保存支持」の宣言条件（CI ⊂ [−ε, +ε]）は下限 −0.242 < −0.15 により不成立。
- **データの重心は微減**: γ̂ = −0.12 は d_model 8 倍で k\* が 0.78 倍（2 割減）に相当。

## 5. 副判定と対照

- **副判定 1（系列別符号一致）: 成立**。γ_qwen = −0.140±0.072（4 点）・γ_gemma = −0.110±0.076（4 点）——**独立二系列で「負・同水準」が再現**。gemma は 12b の追加（k\*=6.70・系列内最低）で前回のゼロ近傍（−0.006）から −0.11 へ動いた——上端の点が系列内トレンドを確定させる、という 80GB 取得判断の狙いどおりの情報だった。
- **副判定 2（k\*・erank・PR の 3 測定器符号一致）: 未実施**——erank/PR の operationalization（KV-CoRE 式の一次照合）が未了。**「保存 vs 微減」の最終宣言はこの副判定を待つ**（k\* 単独の微減は lens 系測定器の系統誤差の可能性を排除しきれないため——§6）。
- **対照**: rand-orth = 9/9 モデルで実質ゼロ（測定器下限チェック成立・Methods 直行）。logit-lens paired = **Qwen 5/5 モデルで 10/10 全 seed 正**（+1.4〜+5.2 語）／**Gemma は 270m ±0・1b 1/10・4b 1/10・12b 0/10（−2.52 語）**——系列非対称が 4 点目（12b）で最強の形で再現。12b の band 深度（48-77%）は gemma-4b（50-79%)と同水準なので、**深度単独仮説では 12b の逆転の強さを説明しきれない**——「Gemma 系では J-lens の付加価値が系統的に負」という系列レベルの事実として報告し、機構（lens fit 品質・アーキテクチャ・トークナイザ）の切り分けは open。

## 6. 解釈と限界（報告の言葉）

**言えること**:
1. J-space 容量（語彙化可能な放送チャネルの実効本数）は、**d_model 640→4096・params 0.27B→12B の 2 系列 8 モデルで増えない**（H_G 棄却・確定）。superposition 系の「総表現容量 ∝ ニューロン数」から派生する成長予想は、この操作化では実現していない。
2. 点推定は**微減（γ ≈ −0.12）**で、k 大・幅統一・端点除去の全感度方向で負が強まる。「固定和」仮説はデータ上「固定どころか希釈」側に倒れつつある——サイズとともに総表現空間は広がるのに、語彙で読み出せる放送チャネルはむしろ薄まる、という絵。
3. 判定枠上の出口は「決定不能・実額報告」（事前登録どおり）。**「微減の確定」を宣言しないのは**: (a) CI が僅かにゼロを含む（主判定 k=64）(b) 副判定 2（独立測定器）が未実施 (c) lens 品質のサイズ依存性という系統誤差候補が残る、の 3 点による。

**系統誤差の候補（微減が人工物である経路）**:
- **lens fit 品質のサイズ依存**: 大モデルほど既製 lens が相対的に under-fit なら人工的な微減が出る。fit 分散は 0.6B で無視可能と確定済み（E3）だが、fit **品質**のサイズ依存は未統制。反証材料 = 収束後 identity_distance のサイズ依存を lens config から集計する（軽作業・次工程でやる価値あり）。
- band 幅・深度の残余交絡: 幅は band4 感度で方向を確認済み（微減を隠す側）。深度は系列内でほぼ一定。
- リスト語彙の series 依存（single-token canon がトークナイザ依存）: 系列内傾きには効かない——系列間で符号が一致しているので一次影響は否定的。

## 追記 3（2026-07-11 研究者）: gemma-3-27b（H200）の算入と更新判定 — 測定器依存の顕在化

H200 セッション（エンジニア実行・実行ログに記録）で gemma-3-27b（d=5376・Gemma 上端）を取得。**band 規則は健全**（被覆率 30/61 = 49.2%——監視条項の 90% を大きく下回る・深度 47-79% は他 Gemma と同水準・14b 型の故障は不再現）、**定常性規約成立**（|B4−B2| = 0.70）、除外理由なし——**main 回帰に算入**（取得前に「取得したら Gemma 5 点で γ 更新」を宣言済み・恣意的追加ではない）。rand-orth = 0.328（seed0 のみ 0.95）は既存レンジよりやや高いが、band 幅 20 層（全モデル最大）では band-min の床が幅とともに上がる構造があり、かつ J-lens 値の 7.5% で分離は維持——「実質ゼロ」判定は維持しつつ幅依存の床を開示。
